# Smart_Outlet


## Connect WiFi
To connect with WiFi, create a txt file named, creds.txt. Put the SSID on one line, and the password on another. Save, push to esp, and reboot.

ex:
something_SSID
something_password

## Files needed on ESP
main.py
smart_control.py
websrv.py
index.html
